## 디자인 방향 (선택 — /design 트랙용, v0.8)

> 토큰의 씨앗이 브랜드색 하나였듯, 디자인의 씨앗은 이 섹션입니다.

- **방향 키워드 (2~3개, 구체 명사)**: "여행 잡지 에디토리얼", "보딩패스의 사물성", "필름 사진 톤"
- **레퍼런스 × 차용점 (반드시 짝으로 3개)**:
  - Airbnb → 코랄 브랜드색의 절제 사용 (면이 아닌 점 — 버튼 1개와 숫자에만)
  - 매거진 B → 좌측 정렬 거대 지명 헤드라인 + 메타데이터의 에디토리얼 위계
  - 실물 보딩패스 → 절취선·스텁·구간 표기(ICN→KIX)의 사물 디테일
- **시그니처 요소 (이 제품이 기억될 단 하나, 주제의 사물 세계에서)**: D-day 스텁이 달린 절취선 보딩패스 카드. 예약 화면에선 체크인→체크아웃 구간으로 번역
- **프로젝트 전용 금지 (공통 anti-slop에 추가)**:
  - 검색바를 화면 최상단에 두지 않는다 (이 앱의 일은 탐색이 아니라 "다음 여행 이어가기")
  - 카테고리 아이콘 가로줄 금지 (여행앱 템플릿의 1번 신호)
  - 사진에 필터 없이 원본 노출 금지 (grayscale 0.35 + contrast 1.1 필름 톤 통일)
- **다이얼 (1~10, 비우면 변주7/모션4/밀도4)**: 변주 7 · 모션 3 · 밀도 4
